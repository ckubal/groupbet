# NFL Betting Group App

## The Problem
Group of friends bet on NFL games via text. Need centralized app for:
- Viewing odds (by game time slots: Thursday night, Sunday 10a/1p/night, Monday)
- Voting on games (thumbs up/down system) 
- Seeing group alignment on picks
- Logging actual bets + who's participating
- Live game tracking with real-time scores
- Settlement calculations and payout summaries

## Key Requirements
- **No app download required** (Progressive Web App)
- Easy to share via iMessage group chat
- Real-time updates for all group members
- Simple authentication for small private group
- Support for Bovada and Str8Play betting slips
- Both manual entry and slip parsing capabilities
- Weekend settlement with "who owes who" breakdown

## Betting Platforms
Currently using:
- **Bovada** - Primary platform, structured confirmation slips
- **Str8Play** - Secondary platform, different format

## Group Context
- Small private group (friends betting together)
- Currently coordinate via iMessage
- Want to eliminate text message coordination overhead
- Need easy bet tracking and settlement
